AlphaTex 사용법

AlphaTex는 기타 타브 악보를 텍스트로 표현하는 마크업 언어입니다.

기본 구조

AlphaTex 파일은 메타데이터 헤더 → 점(.) → 음표 데이터 순서로 구성됩니다.

\title "곡 제목"
\artist "아티스트"
\tempo 120

.
// 여기서부터 음표 입력
:4 0.6 2.5 3.4 |
점(.) 은 필수입니다. 메타데이터와 음표 데이터를 구분하는 구분자이며, 없으면 렌더링되지 않습니다.

메타데이터

키워드 설명 예시
\title 곡 제목 \title "My Song"
\subtitle 부제목 \subtitle "Demo"
\artist 아티스트명 \artist "User"
\album 앨범명 \album "Album"
\tempo BPM (곡 중간에도 사용 가능) \tempo 140
\tuning 튜닝 설정 \tuning E4 B3 G3 D3 A2 E2
\capo 카포 위치 \capo 2

음표 표기법

음표는 프렛번호.줄번호 형식으로 표기합니다. 줄 번호는 1번 줄(얇은 줄)이 1, 6번 줄(두꺼운 줄)이 6입니다.

0.6   // 6번 줄 개방 (E)
2.5   // 5번 줄 2프렛 (B)
3.4   // 4번 줄 3프렛
12.1  // 1번 줄 12프렛

기타 표준 튜닝 (줄 번호)

1번
e (얇은)
2번
B
3번
G
4번
D
5번
A
6번
E (두꺼운)

음표 길이

:숫자로 길이를 지정합니다. 한 번 설정하면 변경 전까지 유지됩니다.

기호 길이 예시
:1온음표:1 0.6
:22분음표:2 0.6
:44분음표 (기본값):4 0.6
:88분음표:8 0.6
:1616분음표:16 0.6
:3232분음표:32 0.6
 
// 길이는 변경 전까지 유지됨
:8 0.6 2.6 3.6 5.5 | :4 0.6 :2 3.5 |

코드 · 마디 구분 · 쉼표

기호 설명 예시
( ) 코드 (여러 음 동시에) (0.6 2.5 2.4)
| 마디 구분자 (Bar) 0.6 2.5 | 3.4 5.4
r 쉼표 (Rest) :4 0.6 r 2.5 r
// 한 줄 주석 // 인트로 
// 파워 코드 진행
:4 (0.6 2.5 2.4) (0.6 2.5 2.4) r r |
:4 (3.6 5.5 5.4) (3.6 5.5 5.4) r r |

주요 테크닉 (Effect)

음표 뒤에 중괄호 { }로 효과를 추가합니다.

기호 테크닉 예시
h 해머온 (Hammer-on) 5.2{h} 7.2
p 풀오프 (Pull-off) 7.2{p} 5.2
b (포인트...) 벤딩 (Bend) 7.3{b (0 4)}
s 슬라이드 (Slide) 5.3{s} 7.3
v 비브라토 (Vibrato) 7.3{v}
tr fret duration 트릴 (Trill) 7.2{tr 9 16} 
// 해머온 & 풀오프
:8 5.2{h} 7.2 9.2{p} 7.2 5.2 r r r |

// 벤딩 (0 → 반음 2개 = 1음 올림)
:4 7.3{b (0 4)} |

// 비브라토
:2 7.2{v} |

전체 예시

\title "All Techniques Demo"
\subtitle "AlphaTex Reference"
\artist "MetaVibe"
\tempo 120

.
// ① 음표 길이
:1 0.6 |
:2 0.6 2.5 |
:4 0.6 2.5 3.4 5.4 |
:8 0.6 2.5 3.4 5.4 7.3 5.4 3.4 2.5 |
:16 5.1 7.1 5.1 7.1 5.1 7.1 5.1 7.1 5.1 7.1 5.1 7.1 5.1 7.1 5.1 7.1 |

// ② 쉼표
:4 0.6 r 2.5 r |

// ③ 코드
:4 (0.6 2.5 2.4) (0.6 2.5 2.4) (3.6 5.5 5.4) (3.6 5.5 5.4) |

// ④ 해머온
:8 5.2{h} 7.2 5.2{h} 7.2 5.2{h} 7.2 r r |

// ⑤ 풀오프
:8 7.2{p} 5.2 7.2{p} 5.2 7.2{p} 5.2 r r |

// ⑥ 레가토 슬라이드
:4 5.3{sl} 7.3 5.3{sl} 7.3 |

// ⑦ 벤딩
:4 7.3{b (0 4)} r 9.3{b (0 4)} r |

// ⑧ 비브라토
:2 7.2{v} 9.3{v} |

// ⑨ 트릴
:2 7.2{tr 9 16} 5.2{tr 7 16} |

// ⑩ 페이드인
:2 0.6{f} r |

// ⑪ 트레몰로 피킹
:4 7.3{tp 8} 7.3{tp 8} 7.3{tp 8} 7.3{tp 8} |

// ⑫ 템포 변경 + 속주
\tempo 160
:16 12.1 15.1 12.1 15.1 12.1 15.1 12.1 15.1 12.1 15.1 12.1 15.1 12.1 15.1 12.1 15.1 |

// ⑬ 엔딩
\tempo 80
:1 (0.6 2.5 2.4 0.3 0.2 0.1)
